Pytanie 'Czy wskazany algorytm jest całkowicie poprawny?': 
Pseudokod: 
```c
find(array, len, key) {
    i = 0
    
    while (i < len && array[i] != key) {
      i = i + 1;
    }
    
    if (i + 1 >= len) {
      return -1;
    } else {
      return i;    
    }
}
```
