# Factorial
## Pseudokod i udowodnienie ze jest częściowo poprawny
### Analiza
Dodatkowe zmienne do f = 1 do inicjacji pętli

### Pseudokod
```
f = 1

while(n--){
  if(n != 0) f *= n
}

return f
```
### Pętla
**Q** (warunek wstępny ): f = 1  
**B** (warunek zatrzymania): while(n--)  
**S** (ciało pętli ): f *= n  
**R** (warunek końcowy): \sum_{k=0}^{i-1} arr[k]
